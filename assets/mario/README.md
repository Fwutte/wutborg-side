# Super Mario Bros. sprite sheets

These assets are used for the private, non-commercial version of the game.

Source sheets:

- Mario & Luigi: https://www.spriters-resource.com/nes/supermariobros/asset/50365/
- Enemies & Bosses: https://www.spriters-resource.com/nes/supermariobros/asset/52570/
- Items, Objects and NPCs: https://www.spriters-resource.com/nes/supermariobros/asset/52569/
- Tileset: https://www.spriters-resource.com/nes/supermariobros/asset/52571/

Sheet ripper: Superjustinbros  
Original game assets: Nintendo

The `*-transparent.png` files are local chroma-keyed derivatives of the source
sheets. Their sprite coordinates are defined in `js/pips-skybound.js`.
